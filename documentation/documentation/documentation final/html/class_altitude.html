◆ ReduceAltitude()

void Altitude.ReduceAltitude ( )

A simple loop to show a simple animation of the altitude graphic The function reduces the value airspeed over time. If the value is 0, it is reset to the max airspeed, 30. The value of the airspeed is also sent to the text box to display the current value.

This function is not used when getting data from the listener.
