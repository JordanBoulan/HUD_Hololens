Detailed Description

Author: Grant Ikehara, Kevin Do File: Altitude.cs
Date Last Modified: 2/26/2017
Description: Altitude code to manage the animation and text of the altitude graphic.

The program finds the required game objects ("AltitudeBarFill" and "AltitudeText") in Unity in order to change them. The AltitudeBarFill changes the graphics, and the AltitudeText will display the value of the airspeed.
