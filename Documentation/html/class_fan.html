Detailed Description

File: Fan.cs
Date Last Modified: 3/15/2017
Description: Fan code to manage the fan graphic.

The program locates the required game objects from Unity ("FanFill" and "FanText") in order to change them. The FanFill changes the graphics, and the FanText will display the fan value.

Member Function Documentation

◆ changeColor()

void Fan.changeColor ( )

Changes the color of the fan graphic and text depending on the value of the fan percentage.


Red: when the fan is between 0 and 10 percent.
Yellow: when the fan is between 10 and 25 percent.
Green: when the fan is 25 to 100 percent.
